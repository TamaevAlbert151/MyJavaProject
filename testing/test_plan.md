# План тестирования

## Цели тестирования
- Проверка корректности выполнения всех функций приложения (функциональное тестирование).
- Проверка удобства и восприятия интерфейса пользователями (пользовательское тестирование).

## Типы тестирования
1. **Функциональное тестирование**:
   - Проверка всех функций приложения, таких как создание пользователя, авторизация, удаление данных и т.д.
   - Используемые инструменты: JUnit для автоматических тестов.
   
2. **Пользовательское тестирование**:
   - Оценка интерфейса и удобства работы приложения для конечных пользователей.
   - Используемые инструменты: Google Forms для сбора отзывов.

## Среда тестирования
- Операционная система: Windows 10
- Среда разработки: IntelliJ IDEA
- Язык программирования: Java 11

## Критерии завершенности тестирования
- Все тесты функциональности должны быть пройдены успешно.
- Полученные отзывы пользователей должны быть обработаны и учтены для дальнейшего улучшения приложения.
