# MyJavaProject
Учебный проект на Java — Task Manager

## Описание проекта
Учебный проект на Java, предназначенный для практики программирования.  
Проект представляет собой **консольное приложение Task Manager**, позволяющее пользователю создавать и просматривать список задач.  

### Основные функции:
- Добавление задач
- Просмотр списка задач
- Простая консольная реализация

## Установка и запуск
1. Клонируйте репозиторий:
```git clone https://github.com/TamaevAlbert151/MyJavaProject.git```
```cd MyJavaProject```

2 Скомпилируйте проект:
```javac -d out src/main/java/com/example/*.java```

3. Запустите приложение:  
```java -cp out com.example.App```

4. Примеры использования

После запуска приложение выведет:
Welcome to Task Manager! 
Task list: 
- Learning assignment in Java
